# Google maps coded in Vue/Nuxt

## Demo
Navigate to gmapsgif.gif to view the demo


## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

```
## Env
```
MAP_API_KEY = ""
```
